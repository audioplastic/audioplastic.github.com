I am very fond of the [TextMate 2 open source editor](https://github.com/textmate/textmate). I use it to write posts for this blog. I also really like the 'cobalt' text formatting theme that is included with TextMate.

I do not like the hideous default themes that ship with Xcode, and neither does [this guy](http://www.barowy.net/tips/cobalt.html). Daniel Barowy kindly supplies a cobalt emulation theme for xcode on his site, but the file format does not work with the latest version of XCode. I converted this theme and tweaked it slightly to darken the appearance console output. The screenshot below shows Xcode adjacent to TextMate.

![](http://i45.tinypic.com/2mzayqx.png)

If you like the look of this, then copy and paste the text from the code box below into the following file.

```
~/Library/Developer/Xcode/UserData/FontAndColorThemes/cobalt.dvtcolortheme
```

Finally, restart Xcode and select the Cobalt theme under the preferences menu.
